# List of recipes

It has only two pages:

## List of recipes

  * **URL**
  
    `/`
  
  * **Description**
  
    Just list of recipes
    
  * **URL Params**
  
    `page: [integer]`    
  
## Page of recipe

  * **URL**
  
    `/recipe/:id`
  
  * **Description**
  
    Full description of recipe
    
  * **URL Params**
  
    `id: [string]`
